# E-Commerce Sales Analysis

## Project Overview
This project analyzes E-Commerce sales data to identify business insights related to sales, profit, categories, regions, and order trends.

## Tools Used
- Excel
- SQL (MySQL)
- Python (Pandas, Matplotlib)

## Dataset
The dataset contains 9,994 orders with information about sales, profit, categories, regions, shipping mode, and order dates.

## Project Components

### Excel Dashboard
- Interactive Dashboard
- KPI Cards
- Slicers
- Category and Region Analysis

### SQL Analysis
Performed business analysis using SQL:
- Total Sales
- Total Profit
- Highest Sales Category
- Region-wise Analysis
- Year-wise Orders

### Python Analysis
Used Python for:
- Data Exploration
- Missing Value Analysis
- Business Insights
- Trend Analysis

## Key Insights
- Technology category generated the highest sales.
- West region contributed the highest revenue and profit.
- Furniture had the lowest profitability.
- Order trends showed slight fluctuations over the years.
